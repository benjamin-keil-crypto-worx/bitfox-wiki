EngineBuilder

Class Engine Builder The Engine Builder Class is just a helper class that allows you to be more in control of creating BitFox Engines By Using interface methods that set dedicated fields on the Engine. This class is accessible through Bitfox exports and can be imported into your code base by typyng let {builder} = require("bitfox").bitfox
